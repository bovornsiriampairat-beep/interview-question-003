## 📋 สรุปแนวทางการพัฒนา

> ในการทดสอบนี้เป็นการเชื่อมต่อระบบแบบแยกหน้าบ้านและหลังบ้าน **(Full-Stack Separated Application)**
> โดยได้ทำการปรับปรุงและแก้ไขส่วนต่างๆ เพื่อให้ระบบทำงานร่วมกันได้อย่างสมบูรณ์ตามโจทย์กำหนด

### 1. 🖥️ การแก้ไขโครงสร้างหน้าบ้าน (Angular Frontend)

#### 🔷 Angular Signals Architecture
ปรับเปลี่ยนโครงสร้างคลาสหลักจาก `app.component.ts` เดิม ไปรันผ่านโครงสร้าง `App` ตัวใหม่ (`app.ts`, `app.html`)
และเลือกใช้ **Angular Signals (`signal`)** เพื่อจัดการ State ข้อมูลตาราง
ซึ่งช่วยให้หน้าจอทำการเรนเดอร์ข้อมูลใหม่ได้ทันที *(Reactive Change Detection)*

#### 🔷 Dynamic Environment Config
สร้างไฟล์ตั้งค่าระบบไว้ที่ `public/assets/config.json` เพื่อแยกตัวแปร URL พอร์ตของ API ออกจากโค้ดระบบ
สะดวกต่อการเปลี่ยนสภาพแวดล้อมการทดสอบ

#### 🔷 UI/UX Design
- ออกแบบหน้าเพจให้มีลักษณะ **Generic UI** ไม่ระบุชื่อองค์กรใดๆ เพื่อความเป็นสากล
- ใช้ฟอนต์ **Google Sans** ร่วมกับ Open Sans เพื่อความสวยงามและอ่านง่าย
- ปรับสีปุ่มใน Modal Windows ให้ถูกต้องตามโจทย์

| ปุ่ม | สี | ตำแหน่ง |
|---|---|---|
| ✅ ยืนยัน (อนุมัติ / ไม่อนุมัติ) | 🟢 เขียว | ปกติ |
| ❌ ยกเลิก | 🔴 แดง | ขวาสุด |

---

### 2. ⚙️ การเชื่อมต่อระบบหลังบ้าน (.NET Backend API)

- เปิดทดสอบเซิร์ฟเวอร์ .NET Web API ผ่านพอร์ต `http://localhost:5184`
- แก้ไขปัญหาการบล็อก Cross-Origin Request ระหว่างพอร์ต `:4200` (Frontend) และ `:5184` (Backend)
  โดยเปิดนโยบาย **CORS (Cross-Origin Resource Sharing)** ในไฟล์ `Program.cs`
  ทำให้รับส่ง Data Array ครบทั้ง 10 รายการได้สำเร็จ

## 🚀 ขั้นตอนการรันเพื่อทดสอบระบบ

### Step 1 — รัน Backend Web API

```bash
# เข้าโฟลเดอร์โปรเจกต์หลังบ้าน
cd Com.Example.Approval.Api

# รัน API Server
dotnet run
```

---

### Step 2 — ตั้งค่าและรัน Angular Frontend

**2.1** เปิดไฟล์ `example-approval-ui/public/assets/config.json` และตรวจสอบ URL ของ API:

```json
{
  "apiUrl": "http://localhost:5184"
}
```

**2.2** ติดตั้ง dependencies:

```bash
npm install
```

**2.3** รัน Web Application:

```bash
npx ng serve
```

**2.4** เปิด Browser แล้วเข้าใช้งานที่:

```
http://localhost:4200
```
